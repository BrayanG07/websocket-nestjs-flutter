## Required Package
```
yarn add @nestjs/websockets@^9.0.4 @nestjs/platform-socket.io@^9.0.4
yarn add socket.io@^4.5.1
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
